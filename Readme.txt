1-)Kodu Çalıştırmadan evvel chromedriver.exe dosyasını işletim sistmeinize Path olarak eklemeniz gerekmektedir.
2-)https://selenium-python.readthedocs.io/installation.html başka bir tarayıcı kullanıyorsanız bu adresten uygun tarayıcı driverini indirip path olarak ekleyip kodda Chrome yerine tarayaıcının adını yazarak çalıştırabilirsiniz.
3-)Selenium yazılımını cmd ile veya IDE'deki konsola pip install selenium yazarak indirmeniz gerekmektedir.
4-)Koddaki sayac sayısı kaç dk veri çekileceiğini göstermektedir.Kod dakikada 1 bir veri çekip excele yazar.If değerinde sayac'ın değerini ne kadar yaparsanız ko o kadar çalışır ve o kadar dakika veri çeker.
